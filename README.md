tikzpostertable
===============

Defines a `tikztable` environment for the [tikzposter](https://www.ctan.org/pkg/tikzposter) document class similar to `tikzfigure`.

## Usage

```tex
\begin{tikztable}[Put the table caption here]
	\label{tab:mytable}
	\centering
	\begin{tabular}{}
		...
	\end{tabular}
\end{tikztable}

```
